# Documentation Archive

This directory preserves completed, superseded, and dated documentation. For
current authority, start at the [documentation index](../README.md).

## Policy

- Archive content; do not delete it merely because work completed or direction
  changed.
- Keep original filenames where practical so repository history and searches
  remain useful.
- Treat current specs and later numbered decisions as authoritative when an
  archived document conflicts with them.
- Change archived files only for link repair, provenance notes, or factual
  transcription corrections.
